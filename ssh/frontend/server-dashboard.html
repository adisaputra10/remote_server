Quick Setup Guide

Get ZConnect running in minutes with our easy-to-follow setup process. Download the components and follow the simple installation steps.

1. Server Relay

Central hub for secure connections

Installation:

./zconnect-relay --port=8080 sudo systemctl enable zconnect-relay

Requires: Linux/Windows Server, Port 8080 open

2. Agent

Install on target servers

Installation:

./zconnect-agent --relay=your-relay-url sudo systemctl enable zconnect-agent

Requires: SSH access, sudo privileges

3. Client

Run on your local machine

Usage:

zconnect connect server-name zconnect list

GUI and CLI versions available
