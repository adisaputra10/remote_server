Usage of test-ssh-pty.exe:
  -agent string
    	Target agent ID
  -compress
    	Enable gzip compression
  -insecure
    	Skip TLS certificate verification
  -key string
    	Path to SSH private key file
  -log
    	Enable command logging (default true)
  -log-dir string
    	Directory for SSH session logs (default "pty-logs")
  -password string
    	SSH password (not recommended, use key)
  -relay-url string
    	Relay WebSocket URL (e.g., wss://relay.example.com/ws/client)
  -token string
    	Auth token (or set TUNNEL_TOKEN env)
  -user string
    	SSH username
